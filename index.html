‚ùì Frequently Asked Questions

How do I change the base currency?

1. Go to "Settings" from the main screen

2. Tap on the "Base Currency" section

3. Select "Change" to see all options

4. Search and select your preferred currency

5. Changes will apply immediately throughout the app

How do I add currencies to my favorites?

1. Go to "Settings"

2. Scroll down to "Add Currencies"

3. Use the search bar to find specific currencies

4. Tap the "+" button next to the currency you want to add

5. The currency will appear in your favorites list

Why don't decimal commas work?

This issue has been resolved in recent versions. The app now automatically recognizes both dot (.) and comma (,) based on your regional settings. If you're still having problems:

1. Make sure you have the latest version

2. Restart the application

3. Check your device's regional settings

How does developer mode work?

Developer mode (visible only in DEBUG versions) allows:

  • Toggle between real and simulated data
  • Switch between different API providers
  • Test functionality without consuming API limits

This functionality is available only for testing and development.

How often are exchange rates updated?

Exchange rates are automatically updated every time you open the app or navigate between screens. We use reliable APIs like ExchangeRate-API and OpenExchangeRates to ensure accurate and up-to-date data.

Can I use the app offline?

SnapRates requires an internet connection to get the most recent exchange rates. Without connection, the app will show the last cached rates, but we recommend staying connected for updated data.
